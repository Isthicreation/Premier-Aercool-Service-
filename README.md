# Premier Aerocool – Static Website

This is a bilingual (English/Arabic) mobile‑responsive brochure site built with HTML, TailwindCSS (CDN), and vanilla JavaScript. No build tools or backend.

## Run locally
- Open `index.html` in your browser.

## Deploy
- Netlify: drag‑and‑drop the `site/` folder, or connect a repo with `site/` as root.
- GitHub Pages: push the folder as the repository and enable Pages for `main` branch root.

## Edit content
- Text lives in `index.html` and `app.js` (I18N dictionaries).
- Images are hot‑linked from Unsplash/Pexels/Wikipedia and can be replaced with local files under `assets/images/`.

## Attribution
Stock images courtesy of Unsplash/Pexels. Client logos via Wikipedia (SVG). Replace with your own media when available.
